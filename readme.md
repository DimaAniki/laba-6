Интернет-магазин книг (на Django + Next / Frontend MVP)
1. Общее описание проекта
Интернет-магазин книг «BookNest» — это веб-платформа для онлайн-покупки печатных и электронных книг различных жанров.
Проект ориентирован на любителей чтения, студентов и профессионалов, которые ищут удобный способ выбора и покупки книг.

Название «BookNest» символизирует уют, знания и пространство, где каждая книга находит своего читателя.

2. Цели проекта
Предоставить пользователям удобный интерфейс для поиска и покупки книг

Создать масштабируемую основу для расширения функционала

Обеспечить адаптивность и корректную работу на всех устройствах

Подготовить архитектуру для подключения бэкенда и базы данных
